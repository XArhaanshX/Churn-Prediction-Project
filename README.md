# Customer Churn Prediction Project

This project predicts telecom customer churn using Machine Learning models (Logistic Regression and Random Forest).  
It includes:

- Exploratory Data Analysis (EDA)
- Data Preprocessing & Encoding
- SMOTE class imbalance handling
- Model Training & Evaluation
- Deployment using FastAPI
- Working REST API for Churn Prediction

# How to Run the API
1. Install Dependencies

Make sure you have Python installed (version 3.10+ recommended).

In the project folder, run:

pip install -r requirements.txt


This installs:

FastAPI

Uvicorn

Pandas

NumPy

Scikit-learn

Joblib

Imbalanced-learn (SMOTE)

2. Start the API Server

Run this command in your terminal from inside the project folder:

python -m uvicorn api:app --reload


If your Python command is py, use:

py -m uvicorn api:app --reload


This starts a local server at:

http://127.0.0.1:8000

3. Open Swagger UI (Interactive API Tester)

Go to:

http://127.0.0.1:8000/docs


Swagger UI allows you to:

1)Test /predict-row with JSON input

2)Upload a CSV to /predict-csv

3)View expected input formats

4)See API documentation automatically generated by FastAPI

5. Stop the Server

Press:

CTRL + C


in the terminal to stop the API.
